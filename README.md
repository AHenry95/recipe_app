# Recipe App

A Django web application for managing and sharing recipes. When completed, users will be able to create, view, search for recipes based on ingredients.
The application automatically assigns a dificulty rating to each recipe based on the recipe's cooking time and the number of ingredients.

## Features (Coming soon)
- Create and manage recipes with ingredients, cooking time, and instructions
- Automatic difficulty rating (Easy, Medium, Intermediate, Hard)
- Search recipe by ingredients
- Django admin panel for database management
- User authentheication (login/logout)

## Technologies Used

- Python 3.8+
- Django 4.2
- SQLite (development)
